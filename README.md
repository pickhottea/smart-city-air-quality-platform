# smart-city-air-quality-platform

Air-quality ingestion, standardization, quality control, telemetry, and policy-oriented monitoring for smart-city open data.

## Scope
- Cities: Torre Pacheco, Molina de Segura
- Pollutants: PM2.5, PM10, NO2, O3
- Layers: Bronze, Silver, Gold
- Quality checks: missingness, range checks, duplicate timestamps, stale data, station completeness
- Telemetry: ingestion logs, freshness metrics, quality metrics, run spans
