# smart-city-air-quality-platform

Smart city big-data analysis project focused on air-quality ingestion, standardization, quality control, analytics, and telemetry side outputs.

## Scope
- Cities: Torre Pacheco, Molina de Segura
- Main entity model: AirQualityObserved
- Main layers: raw, silver, gold
- Side outputs: pipeline logs, trace spans, quality metrics, freshness metrics
